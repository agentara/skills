# Health

Medical support tools for symptom triage and lab report interpretation.

- **[emergency-triage](emergency-triage/SKILL.md)** — Help users quickly assess urgent or semi-urgent symptoms through focused triage, likely-cause ranking, department guidance, and suggested hospital checks.
- **[lab-interpreter](lab-interpreter/SKILL.md)** — Interpret medical lab or test reports from images, PDFs, or text by explaining abnormal values, possible significance, and practical next steps in plain language.
